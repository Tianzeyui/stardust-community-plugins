# BrainPlus 社区插件

[BrainPlus](https://github.com/Tianzeyui/brainPlus) 的社区插件集合。

## 插件列表

| 插件 | 描述 |
|------|------|
| `_template` | 插件开发模板，创建新插件时使用 |
| `hello-world` | 入门示例插件 |
| `diary` | 日记插件 |
| `inspiration` | 灵感记录插件 |

## 使用方式

将插件目录复制到 [BrainPlus](https://github.com/Tianzeyui/brainPlus) 的 `plugins/` 目录下即可使用。

## 开发插件

参考 `_template` 目录作为模板，复制后修改即可开始开发。
